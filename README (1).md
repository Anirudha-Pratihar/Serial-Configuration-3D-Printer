## Project Overview

This project presents the design and development of a **Serial
(Cartesian) Configuration 3D Printer**.\
The printer is inspired by the Ender 3 architecture and is designed to
provide a robust structure, a customizable large print bed space, and
precise motion control using Marlin firmware.

------------------------------------------------------------------------

## Objectives

-   Construct a Serial type Cartesian 3D printer\
-   Create a customizable large print workspace\
-   Implement Ender 3-like control system\
-   Ensure structural rigidity and precise printing

------------------------------------------------------------------------

## Mechanical Design

### Frame

-   2020 Aluminium Extrusions\
-   Aluminium flat bars\
-   M4 nuts and screws

### Motion System

-   Linear guides\
-   GT-2 pulley belts (X & Y axes)\
-   Ball screw mechanism (Z-axis)

### Tools Used

-   Hand Drill\
-   Filer\
-   Hacksaw\
-   3D printed prototype parts

------------------------------------------------------------------------

## Electronics Configuration

-   Arduino Mega 2560\
-   RAMPS 1.4 Shield\
-   4 Stepper Motors (X, Y, Z, Extruder)\
-   RepRap LCD Display\
-   Limit switches and Thermistors\
-   Heated bed with heat pad\
-   Prusa-type extruder with cooling fan

------------------------------------------------------------------------

## Control System

-   Marlin Firmware\
-   PID control for motion precision\
-   PID temperature control for heated bed and nozzle\
-   RepRap display with rotary knob for user interface

------------------------------------------------------------------------

## Results
